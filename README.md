# What is this?
librairy to control servo using their pca9685 board + their FT232H GPIO breakout board, in this case a laser turret.

# parts

USB GPIO/I2c board : https://www.adafruit.com/product/2264
PCA9685 board : https://www.adafruit.com/product/815
turret frame : https://www.adafruit.com/product/1967
